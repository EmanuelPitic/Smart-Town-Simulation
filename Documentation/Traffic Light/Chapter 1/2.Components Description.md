## 2. Components Description

### To achieve this project, carefully selected hardware components were used, each having a specific role in the operation and connectivity of the intelligent traffic light system:

### 1. Raspberry Pi Pico W ("Pico 2W")
#### This is the central control unit of the system, responsible for executing the main code, managing peripherals, and handling communications[cite: 44, 46]. The board provides Wi-Fi connectivity, allowing the transmission and reception of MQTT messages to and from the external server[cite: 34, 47]. At the same time, it directly controls signals for the traffic light LEDs, processes sensor data, and manages UART communication with the HC-05 Bluetooth module[cite: 18, 42].

---

### 2. HC-05 Bluetooth Module
#### The HC-05 module is connected to the Pico W via UART and allows the reception of commands from a Bluetooth device (e.g., phone or laptop). It provides the possibility for local/manual control over the traffic light, independent of the Wi-Fi network or the MQTT server, making it useful for rapid configuration or testing[cite: 42].

---

### 3. Traffic Light LEDs
#### Sets of LEDs (green, yellow, red for each direction) are controlled by the Pico W through digital pins[cite: 18]. These reflect the current state of the traffic light for each direction of travel, and the operating logic (manual, sequential, or intelligent) is implemented in software according to the code running on the microcontroller[cite: 11, 47].

---

### 4. Infrared Sensors
#### Infrared sensors detect the presence of vehicles near the traffic light[cite: 46]. The data collected by these are used in the intelligent operating mode to adapt the traffic light durations based on traffic[cite: 44]. The Pico W constantly reads the state of the sensors and adjusts the traffic light cycles based on the implemented algorithm[cite: 46, 47].

---

#### Each component is connected directly or indirectly to the main Raspberry Pi Pico W board, and the interaction between them enables a road traffic control system with remote monitoring and control functionalities[cite: 42, 47].
