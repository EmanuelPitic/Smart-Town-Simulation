## 1. Introduction

### This project aims to develop an intelligent traffic light system based on the Raspberry Pi Pico W board (revision 2, labeled "Pico 2W"). It manages all critical functions of the traffic light: LED control, reading sensor data, UART communication with the Bluetooth module, and interaction with an external server via the MQTT (Message Queuing Telemetry Transport) protocol. The main goal is to ensure efficient traffic control in an intersection, with the ability to switch between three operating modes: manual, sequential, and intelligent, adapted to traffic situations.

### The Raspberry Pi Pico 2W represents an efficient and compact solution based on the RP2350 microcontroller. Thanks to its Wi-Fi connectivity, low power consumption, and compatibility with various sensors and actuators, this board ensures the autonomous operation of the traffic light. Choosing the Pico 2W offers an optimal balance between performance, cost, and flexibility for modern embedded applications.

### The system's operating modes are:
- **Manual:** The traffic light stays green in the specified direction based on commands received locally (via UART/Bluetooth) or remotely (via MQTT).
- **Sequential:** The traffic light operates based on preset times controlled by the user via UART or MQTT messages, alternating intersection access between directions.
- **Intelligent:** Sensors detect the presence of vehicles; the traffic light maintains green in the direction where cars are detected. If there are requests from multiple directions, the system alternates sequentially until traffic flows smoothly.

### Communication via MQTT provides a flexible and scalable architecture, ideal for IoT applications. The system can transmit real-time information, such as vehicle detection or the current state of the traffic light, to the server, facilitating centralized monitoring and analysis.

### By integrating detection, wireless communication, and automated control, the project illustrates a practical application of a cyber-physical system. It highlights how hardware and software can be efficiently combined into a modern embedded solution with real-world applications in urban traffic management.
