## 3. Connecting Components

### The electronic schematic below illustrates how all the main components of the intelligent traffic light system are connected to the Raspberry Pi Pico 2W:

![Traffic Light Circuit Diagram](/Documentation/Traffic%20Light/Images/traffic_light_circuit_diagram.png)

---

### Schematic Explanation

#### 1. **Traffic Light LEDs**

- Three LEDs are used for each direction: red, yellow, and green, connected through a 331 Ω resistor for protection.
- Each LED is directly controlled by a GPIO pin from the Raspberry Pi Pico 2W.
- This configuration is repeated for all four directions/traffic lights in the intersection.
- By selectively lighting the LEDs, the real operation of a traffic light is simulated.

#### 2. **Infrared (IR) Sensors**

- Each IR sensor has three connections: GND (ground), VCC (power), and DATA (signal).
- The DATA pin is connected to a GPIO pin on the Pico 2W, allowing it to read the presence of cars in each direction.
- The IR sensors detect if there are vehicles waiting at the traffic light, and this information is used for the intelligent operating mode.

#### 3. **HC-05 Bluetooth Module**

- The HC-05 module is connected to the Pico 2W board to allow traffic light control via Bluetooth.
- The essential connections are:
  - VCC (5V)
  - GND (ground)
  - TXD and RXD (for UART communication)
- A voltage divider (two resistors of 2 kΩ and 1 kΩ) is used between the module's TXD and the Pico's RXD to reduce the voltage to a level compatible with the Pico (3.3V).
- The STATE pin can be used to indicate the status of the Bluetooth connection.

#### 4. **Raspberry Pi Pico 2W**

- This represents the system's command center. Each GPIO pin either controls an LED or receives a signal from an IR sensor.
- The connections for each LED and each IR sensor are clearly marked on the schematic.
- The board handles:
  - Lighting the corresponding LEDs (red/yellow/green) for each direction.
  - Reading the status of the IR sensors.
  - Communicating with the HC-05 Bluetooth module (via UART and the voltage divider).
  - Transmitting data to a server if connected to Wi-Fi.

---

### Operating Summary

- **The Traffic Light** (LEDs) receives signals directly from the Pico, which determines which color is active in each direction.
- **The IR Sensors** inform the Pico when a car is detected, which influences the automatic or manual traffic light cycles.
- **The HC-05 Module** allows the user to send commands to the traffic light via Bluetooth using phone or laptop applications.
- **All connections** are made directly to the pins of the Raspberry Pi Pico 2W board, according to the schematic.
