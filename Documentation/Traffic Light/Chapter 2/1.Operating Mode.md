## 1. Operating Mode

The intelligent traffic light system allows for remote or local control and monitoring, offering multiple usage options for efficiently managing road traffic.

### How does the system work?

1. **System Startup** Once the system is powered, the traffic light starts automatically and begins operating according to a preset mode.

2. **Local Control via Bluetooth** The user can control the traffic light remotely using a Bluetooth-compatible device (phone or laptop) and a "Bluetooth Terminal" type application.  
   By sending text commands to the system, one can change the traffic light's operating mode or its parameters.

---

### Format of command packets sent via Bluetooth

For the system to understand the commands, each message must be written **on a single line** and follow the exact format below (followed by Enter):

- **Manual Mode:** To set the traffic light to manual control, choose the direction that will receive green:  
  - `m d1`  → Manual mode, direction 1 (the first direction is green)
  - `m d2`  → Manual mode, direction 2 (the second direction is green)

- **Sequential Mode:** To set the automatic mode with custom cycles, use:  
  - `s d1 r 7 y 2 g 5`  
    - `s`  = sequential mode  
    - `d1` = start direction  
    - `r 7` = red 7 seconds  
    - `y 2` = yellow 2 seconds  
    - `g 5` = green 5 seconds  
  (Values can be modified, e.g., `r 10 y 3 g 8`)

- **Intelligent Mode:** For intelligent mode, send the duration for the green color:  
  - `i 5` → intelligent mode, 5 seconds green for the detected direction  
  (The number can be any positive value, according to preference)

---

#### **Practical Example:** If you want the traffic light to operate in manual mode with green in direction 1, type in the Bluetooth Terminal app: `m d1` 
and send the command.

---

### Recommendations for sending commands:
- Write the exact format exemplified above without additional characters.
- Each command must be followed by Enter (or "Send").
- If you send an incorrect command, the system will respond with an error message.

---

### In short:
You can control the traffic light locally via Bluetooth by sending short text commands. Each command corresponds to an operating mode or a specific action and must respect the format presented above.
