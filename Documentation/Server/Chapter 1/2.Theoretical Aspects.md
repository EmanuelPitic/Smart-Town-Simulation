## What is a protocol? 

A protocol is a set of rules and instructions that allow computers to communicate and transmit data packets to each other. As a simple comparison, it is similar to a handshake gesture before starting a conversation – following specific rules of interaction.



## What is MQTT?

MQTT is a message transport protocol using the Client-Server model, based on the publish/subscribe mechanism. Due to its simplicity, resource efficiency, and ability to function in low-signal conditions, it is ideal for various situations, including constrained environments such as machine-to-machine (M2M) communication and IoT projects that require minimal code. Currently, many famous applications adopt this protocol, including Facebook Messenger and Instagram.

To better understand this protocol, we will analyze several key concepts: 

* **Publish / Subscribe** * **Broker** * **Topics** * **Mechanisms such as:** 1. Keep Alive 
    2. Last Will and Testament 
    3. Quality of Service (QoS)
